## 回归 Bug 列表

### Bug #1: VITE_AMAP_KEY 环境变量未定义（低严重程度）

- **严重程度**: 低
- **描述**: 构建时输出警告 `%VITE_AMAP_KEY% is not defined in env variables found in /index.html`
- **影响范围**: 仅构建警告，不影响功能（地图功能使用 Leaflet 而非高德地图）
- **修复建议**: 在 `.env` 文件中添加 `VITE_AMAP_KEY=your_key`，或在 `index.html` 中移除未使用的变量引用
